### Installing

* You can use IDLE, Pycharm or CMD to run this program!!
1. Go to Code
2. Download as ZIP
3. Extract to any folder
4. Open your `cmd` and `cd` the folder you extracted
5. Run `pip install -r requirements.txt`
6. Double Click or Run `gamehub.py`
7. Enjoy!!
